Security Features

  • Domain Restriction: Only wallacemuseum.com and localhost can access images
  • Browser Detection: Allows legitimate browser requests, blocks automated tools
  • CORS Protection: Proper cross-origin headers for web security
  • Caching: Optimized headers for fast image delivery
